Send Rocky a voice message and get back a clean transcript, a summary, and any action items — in seconds.

## What it does

WhatsApp voice notes are downloaded, transcribed using a speech-to-text pipeline, and then summarised by Rocky. Long recordings become short, structured notes. Action items are extracted automatically.

## Key capabilities

- **Full transcription** — supports OGG, MP3, WAV, and WhatsApp's native audio format
- **Smart summary** — extracts the key points in 3–5 bullets
- **Action item extraction** — identifies tasks and follow-ups from the recording
- **Works as input** — you can issue commands to Rocky by voice instead of typing

## How to use it

Send a voice message to Rocky. No command needed — it transcribes and summarises automatically.

Or: *"Transcribe that last voice note and create a task for each action item"*
